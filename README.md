# MobArenaSigns
Easily create Mob Arena Join and Exit signs! For Spigot 1.11

# Description
Allows the creation of two signs:

[Join Arena]

Arena Name


and


[Exit Arena]


These will make the player essentially type /ma join arena and /ma leave
